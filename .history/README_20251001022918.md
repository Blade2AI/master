## Remote Access Bundle on D:\

To create a portable remote-access bundle on D:\, run the deploy script from PowerShell:

- Script: `scripts/deploy_remote_access_bundle.ps1`
- Default target: `D:\RemoteAccessBundle`
- Optional zip: add `-Zip -ZipPath 'D:\RemoteAccessBundle.zip'`

After deploying, open `D:\RemoteAccessBundle` in VS Code and “Reopen in Container”, then run the task “Console: Start TLS (localhost:7681)”.
